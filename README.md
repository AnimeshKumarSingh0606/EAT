# JBOSS-EAP ADDITIONAL TESTSUITE
An additional JBOSS-EAP testsuite in order to facilitate QE

HOW TO USE:
1. Make sure that JBOSS_HOME is set with the path to your JBOSS EAP directory.
2. Run the additional testsuite.
